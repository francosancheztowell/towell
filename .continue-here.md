## Hallazgos ya identificados (patrones duplicados)

### 1. Observer dispatcher pattern (×6+ archivos) — EL MÁS FRECUENTE
```php
$dispatcher = ReqProgramaTejido::getEventDispatcher();
ReqProgramaTejido::unsetEventDispatcher();
// ... trabajo ...
ReqProgramaTejido::setEventDispatcher($dispatcher);
ReqProgramaTejido::observe(ReqProgramaTejidoObserver::class);
```
Aparece en: DragAndDrop, Eliminar(×3), Duplicar, Vincular, Balancear.
En los catch: `observe(ReqProgramaTejidoObserver::class)` sin restaurar $dispatcher — inconsistente.

### 2. Regenerar líneas via observer (×5+ archivos)
```php
$observer = new ReqProgramaTejidoObserver();
foreach ($ids as $id) {
    if ($r = ReqProgramaTejido::find($id)) $observer->saved($r);
}
```
Aparece en: Eliminar(×3), DragAndDrop, Balancear, Vincular.
Variante con `whereIn` en DragAndDrop (más eficiente).

### 3. recalcularFechasSecuencia + bulk update (×4 archivos)
```php
[$updates, $detalles] = DateHelpers::recalcularFechasSecuencia($registros, $inicioOriginal);
foreach ($updates as $idU => $data) {
    DB::table(ReqProgramaTejido::tableName())->where('Id', $idU)->update($data);
}
```
Aparece en: Eliminar(×3), DragAndDrop, EliminarConOrdCompartida(×2).
