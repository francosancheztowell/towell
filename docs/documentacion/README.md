# Documentacion del proyecto Towell

Este repositorio ahora cuenta con dos niveles de documentacion para distintos perfiles de lectura.

## 1. Version corporativa y funcional

Dirigida a direccion, jefaturas, coordinacion operativa, mejora continua, capacitacion y usuarios no tecnicos.

Ruta: `docs/documentacion-corporativa/`

Contenido principal:

- descripcion ejecutiva por fase
- objetivo del proceso
- areas usuarias
- procesos principales
- entradas y salidas
- dependencias entre modulos
- riesgos operativos
- indicadores sugeridos
- diagramas de entendimiento rapido

## 2. Version tecnica para desarrolladores

Dirigida a desarrollo, soporte tecnico, mantenimiento y analisis funcional-tecnico.

Ruta: `docs/documentacion-tecnica/`

Contenido principal:

- rutas y controladores
- funciones principales
- archivos clave
- relaciones entre modelos, servicios y vistas
- anexos tecnicos
- matriz de trazabilidad de rutas

## Recomendacion de uso

- Si el lector busca entender que hace el sistema desde negocio, iniciar en `docs/documentacion-corporativa/README.md`
- Si el lector necesita mantener o evolucionar el sistema, iniciar en `docs/documentacion-tecnica/README.md`
