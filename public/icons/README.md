# Iconos PWA

Este directorio debe contener los siguientes iconos para que la PWA funcione correctamente:

## Iconos requeridos:

1. **icon-192.png** (192x192 píxeles)
   - Icono estándar para PWA
   - Usado en la pantalla de inicio en Android

2. **icon-512.png** (512x512 píxeles)
   - Icono de alta resolución para PWA
   - Usado cuando se instala la app

3. **maskable-512.png** (512x512 píxeles)
   - Icono "maskable" para Android
   - Debe tener padding de al menos 10% en cada lado
   - El contenido debe estar centrado y ser seguro para recortar

## Recomendaciones:

- Usa el logo de tu aplicación como base
- Los iconos deben ser cuadrados
- Fondo sólido o transparente
- Colores contrastantes para buena visibilidad
- Para el maskable: deja espacio seguro de ~40px desde los bordes

## Herramientas para generar iconos:

- PWA Asset Generator: https://www.pwabuilder.com/imageGenerator
- RealFaviconGenerator: https://realfavicongenerator.net/
- Android Asset Studio: https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html

## Nota:

Una vez que agregues los iconos, la PWA estará lista para instalarse. Los usuarios podrán agregar la app a su pantalla de inicio desde el navegador.
